API Description - v1

This page details the application programming interface (API) that powers the Interactive 2012 State Water Plan website. The API is CORS-enabled so that JavaScript clients can make use of the resources it exposes. All water volumes given are in units of acre-feet/year. Unless otherwise specified, all methods return JSON responses.

For any questions regarding the data or how to apply it, please contact: WRPdatarequests@twdb.texas.gov

The following lists describe all methods of the Interactive 2012 State Water Plan website API.
The base URL for all API paths is http://texasstatewaterplan.org/api/v1.

Places

  • /places/regions - list of all valid Regional Water Planning Area letters
  • /places/regions.topojson - TopoJSON representation of Regional Water Planning Areas
  • /places/counties - list of all valid County names
  • /places/counties.geojson - GeoJSON representation of Counties
  • /places/county/COUNTY_NAME.geojson - GeoJSON representation of COUNTY_NAME

Entities

  • /entity - all State Water Plan Entities with their ids, names, and locations (lat/lon)
  • /entity/ENTITY_ID - information about the Entity with the given ENTITY_ID
  • /entity/ENTITY_ID/summary - all Demands, Needs, Water Supply, and Strategy Supply data for the Entity with the given ENTITY_ID
  • /type/entity - list of Entity Water Use Types

Needs

  • /needs - all Needs data
  • /needs/summary - Regional summary of Needs data by Water Use Type
  • /needs/county/COUNTY_NAME - Needs data for COUNTY_NAME
  • /needs/region/REGION_LETTER - Needs data for the Regional Water Planning Area specified by REGION_LETTER
  • /needs/entity/ENTITY_ID - Needs data for the Entity specified by ENTITY_ID
  • /needs/type/WATER_USE_TYPE - Needs data for the Entities with the specified WATER_USE_TYPE

Demands

  • /demands - all Demands data
  • /demands/summary - Regional summary of Demands data by water use type
  • /demands/county/COUNTY_NAME - Demands data for COUNTY_NAME
  • /demands/region/REGION_LETTER - Demands data for the Regional Water Planning Area specified by REGION_LETTER
  • /demands/entity/ENTITY_ID - Demands data for the Entity specified by ENTITY_ID
  • /demands/type/WATER_USE_TYPE - Demands data for the Entities with the specified WATER_USE_TYPE
